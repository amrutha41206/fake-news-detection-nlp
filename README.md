# Fake News Detection using NLP

## Overview
This project detects fake news articles using Natural Language Processing and Machine Learning.

## Dataset
Fake and Real News Dataset from Kaggle.

## Technologies Used
Python
Scikit-learn
TF-IDF Vectorization
Logistic Regression
Google Colab

## Workflow
1. Data preprocessing
2. Exploratory data analysis
3. Feature extraction (TF-IDF)
4. Model training
5. Model evaluation
6. Prediction system

## Results
The model achieved around 98% accuracy in detecting fake news.
